### 基本操作

```
与（&）
或（|）
非（~）
异或（^）
左移（<<）
无符号右移位（>>>）
带符号右移（>>）
```

### 获得最右侧（低位）的 1

```
n & (~n + 1)
```

### 移除最右侧（低位）的 1

```
n & (n - 1)
```

或

```
n - n & (~n + 1)
```

### 将最右侧（低位）的 0 变为 1

```
n | (n+1)
```
