### 在线任务编排框架

####  `Job`执行过程及其对应显示
#### 用户操作映射
| 用户操作 | `Job`状态 | 页面显示 |
| -------- | ------- | -------- |
| 删除操作 | 无数据  | ——       |
| 停止操作 | 无数据  | 已停止   |
| 激活操作 | `READY`   | 准备中   |

#### Job状态映射
| `Job`状态 | `Job`状态解释 | 页面显示 |
| --------- | ------------- | -------- |
| `READY`   | `Job`执行结束 | 准备中   |
| `RUNNING` | `Job`正在运行 | 正在运行 |
| `STOP`    | `Job`异常停止 | 异常停止 |
| ——        | `Job`不再运行 | 已停止   |
