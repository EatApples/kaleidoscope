### Eureka 的设计理念
（1）服务实例如何注册到服务中心：REST 接口

（2）服务实例如何从服务中心剔除：使用心跳

（3）服务实例信息的一致性问题：CP+对等复制

### Zuul 的路由通配符映射规则
| 规则 | 释义                     | 示例             |
| ---- | ------------------------ | ---------------- |
| /**  | 匹配任意数量的路径与字符 | /a/b,/a/b/c,/abc |
| /*   | 匹配任意数量的字符       | /a,/ab,/abc      |
| /?   | 匹配当个字符             | /a,/b,/c         |

### Zuul 工作原理
核心逻辑是 Filter 责任链，能够在进行 HTTP 请求或者响应的时候执行相关操作。
